# Architecture diagram (where it lives)

The authoritative architecture diagram now lives in `architecture.mermaid.md` in this folder and in the Docusaurus admin guide at `docs-site/docs/admin/architecture.md` (with the standalone partial at `docs-site/docs/admin/_architecture-diagram.md`).

The `architecture.excalidraw` file and its exported `architecture.png` are kept for historical reference only and may be out of date. Do not treat them as canonical. When the architecture changes, update `architecture.mermaid.md` and the Docusaurus admin guide; the Excalidraw file does not need to be maintained.
