# Example Documents

Sample packing slips for testing the Receiving module's AI extraction feature.

## How to Use

1. Open one of the packing slip HTML files in a browser
2. Take a screenshot or print to PDF
3. In the app, go to **Receiving > Start Receiving**
4. Select a purchase order
5. Upload the screenshot/photo of the packing slip
6. The AI will extract the order number, vendor, and line items
7. Proceed to tag each asset with a barcode scanner or manual entry

## Files

- `packing-slip-001.html` - Dell Technologies order (3 line items: laptops, monitors, desktops)
- `packing-slip-002.html` - Cisco Systems order (2 line items: switches, access points)

## Tips

- The AI extraction works with photos of printed documents, screenshots, and even handwritten notes
- For best results, ensure the image clearly shows the order number and line items
- The system handles various packing slip formats automatically
