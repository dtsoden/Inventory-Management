Asset Tag Barcodes - SAMPLE-PO-9009 (HP Inc.)

Each card has a Code 128 barcode (1D) and a QR code (2D). Scan either format.

Scanning tips:
  • Set screen brightness to maximum
  • Hold scanner 4-6 inches from screen
  • Try the QR code if the barcode doesn't scan
  • Zoom in (Ctrl+Plus) to make codes larger if needed
  • Print this page for the most reliable scanning
