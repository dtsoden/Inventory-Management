Asset Tag Barcodes - Dell Order (SAMPLE-PO-9005)

Each card has a Code 128 barcode (1D) and a QR code (2D). Your Netum C750 can scan either format.

Scanning tips for screen reading:
  • Set screen brightness to maximum
  • Hold scanner 4-6 inches from screen
  • Try the QR code if the barcode doesn't scan (2D codes are more reliable from screens)
  • If neither scans, zoom in the browser (Ctrl+Plus) to make the codes larger
  • Alternatively, print this page for the most reliable scanning
